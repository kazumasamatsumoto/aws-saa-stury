# AWS Budgets

・AWS の予算管理および使用量の監視サービス

・予算の閾値を超える場合、アラートを発報

・Budgets ダッシュボードで予算及び使用状況の監視

・コスト、使用量、予約、Saving Plans に対する予算の設定

・タグやリソースなどでフィルタリングして分析

・アラート通知の種類：

- E メール通知
- SNS トピックを介した通知
- AWS Chatbot による Slack 通知

・自動アクション機能：

- 予算超過時に特定の IAM ポリシーを適用
- EC2 や RDS インスタンスの起動制限
- 予算超過時のリソース作成制限

・予算テンプレートの利用が可能：

- ゼロスペンド予算（支出ゼロ目標）
- 月次コスト予算
- Savings Plans 使用率予算
