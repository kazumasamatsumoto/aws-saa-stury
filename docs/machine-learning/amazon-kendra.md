# Amazon Kendra

Amazon Kendra は、機械学習を活用したインテリジェントな企業向け検索サービスです。

## 概要

- 自然言語処理（NLP）を使用して、企業のドキュメントやデータから正確な回答を提供
- 非構造化データと構造化データの両方を検索可能
- 質問応答機能により、単なるキーワード検索以上の結果を提供

## 主な特徴

### インテリジェント検索

- 自然言語による質問に対して直接的な回答を提供
- コンテキストを理解し、関連性の高い検索結果を表示
- 機械学習モデルが継続的に学習し、検索精度が向上

### データソースの統合

- SharePoint, Google Drive, Salesforce 等の一般的な企業システムと連携
- S3, RDS 等の AWS サービスとのネイティブ統合
- カスタムデータソースコネクタによる拡張可能性

### セキュリティとコンプライアンス

- きめ細かなアクセス制御
- ユーザー認証との統合
- 暗号化とセキュリティ基準への準拠

## ユースケース

1. 社内ナレッジベース検索

   - 社内文書、マニュアル、FAQ の検索
   - 従業員の生産性向上

2. カスタマーサポート

   - FAQ 自動応答
   - サポート担当者の回答支援

3. 研究開発支援
   - 技術文書の検索
   - 特許情報の分析

## 料金

- インデックス作成されたドキュメント数
- 検索クエリ数
- ストレージ容量

に基づいて課金されます。

## 始め方

1. Amazon Kendra コンソールにアクセス
2. インデックスの作成
3. データソースの設定
4. コネクタの構成
5. 同期スケジュールの設定

## ベストプラクティス

- 定期的なインデックス更新の設定
- 適切なアクセス権限の設定
- 同義語辞書の活用
- カスタム調整の実施

## 制限事項

- 言語サポートの制限
- インデックスサイズの制限
- クエリ制限
- データソース接続数の制限
