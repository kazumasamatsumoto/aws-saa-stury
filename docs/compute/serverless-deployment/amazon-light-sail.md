# Amazon Lightsail

Amazon Lightsail は、AWS が提供する軽量なクラウドサービスで、シンプルな仮想プライベートサーバー（VPS）を手軽に利用できるサービスです。

## 特徴

- 予測可能な低価格の月額料金
- シンプルな管理画面
- 事前設定済みのアプリケーションやスタック（WordPress, LAMP 等）
- 簡単なセットアップと管理
- EC2 と比べて設定項目が少なく、初心者に優しい

## ユースケース

- 小規模な Web サイトやブログ
- 開発環境やテスト環境
- 小規模なアプリケーション
- プロトタイプの開発

## 主な機能

### インスタンス

- Linux/Unix ベース、Windows ベースのインスタンス
- 複数のインスタンスサイズ（CPU、メモリ、SSD）
- 静的 IP アドレス

### データベース

- MySQL データベース
- PostgreSQL データベース

### ネットワーキング

- ファイアウォール設定
- DNS 管理
- ロードバランサー

### ストレージ

- ブロックストレージ
- オブジェクトストレージ

## EC2 との比較

| 機能               | Lightsail                  | EC2                        |
| ------------------ | -------------------------- | -------------------------- |
| 価格体系           | 定額制                     | 従量制                     |
| 設定の複雑さ       | シンプル                   | 詳細な設定可能             |
| スケーラビリティ   | 限定的                     | 高い                       |
| ターゲットユーザー | 初心者、小規模プロジェクト | 上級者、大規模プロジェクト |

## 制限事項

- リージョン間でのリソース移動が制限される
- 高度な AWS サービスとの統合が限定的
- オートスケーリング機能がない
- カスタマイズ性が制限される

## 料金

- インスタンスのスペックに応じた固定月額料金
- データ転送量に応じた追加料金
- 追加のブロックストレージやスナップショットに対する料金

## まとめ

Amazon Lightsail は、シンプルな要件を持つプロジェクトや、AWS を始めたばかりのユーザーに最適なサービスです。複雑な設定を必要とせず、予測可能な料金で利用できる点が特徴です。ただし、プロジェクトが成長してより高度な機能が必要になった場合は、EC2 などの他の AWS サービスへの移行を検討する必要があります。
